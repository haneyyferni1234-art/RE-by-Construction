# Representation Identifiability Under Architecture-Conditioned Symmetry

**Title:** Optimization Equivalence Does Not Imply Representational Equivalence:
Architecture-Conditioned Identifiability Phase Geometry

## Compilation

Requires LaTeX (TeX Live 2020+ recommended):

```bash
# Install LaTeX if needed (Ubuntu/Debian)
sudo apt install texlive-latex-extra texlive-bibtex-extra

# Install LaTeX if needed (Fedora)
sudo dnf install texlive-scheme-medium

# Generate figures (requires python3 + matplotlib + numpy)
make figures

# Compile the paper
make paper

# Or both at once
make all

# Clean
make clean     # remove aux files
make cleanall  # remove aux files + output
```
